# Installing garn

Install `garn` with:

<BashSnippetToCopy >
  curl --proto '=https' --tlsv1.2 -sSf https://garn.io/install.sh | bash
</BashSnippetToCopy>

`garn` needs [`nix`](https://nixos.org/) to be installed, so -- if you don't have nix already -- the above installer
will install nix first, after asking for confirmation.
